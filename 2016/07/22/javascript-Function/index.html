静态成员与实例成员的概念

静态成员和实例成员这两个概念其实也是从面相对象的编程语言中引入的，对应到 JavaScript 中的理解为：

静态成员

  • 静态成员是指静态属性和静态方法，所谓静态，就是有构造函数提供的。

实例成员

  • 实例成员是值实例属性和实例方法，所谓实例，就是由构造函数创建出来的对象。
